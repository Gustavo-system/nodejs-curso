### NMV

instalar nvm
```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.0/install.sh | bash
```

- Uso de nvm
instalar una nueva version de node en nvm
```
nvm install <version-node>
```
cambiar la version de node

```
nvm use <version-node>
```
